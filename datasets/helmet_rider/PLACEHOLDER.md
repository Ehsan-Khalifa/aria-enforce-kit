# Put the helmet dataset here

Download this Roboflow dataset (YOLOv8 export) and extract it into THIS folder:

**Helmet and No Helmet Rider Detection (v5, "Balanced Version-Augmented")**
https://universe.roboflow.com/gw-khadatkar-and-sv-wasule/helmet-and-no-helmet-rider-detection
- Classes: With Helmet / Without Helmet / licence  (3 classes)
- License: CC BY 4.0
- On Roboflow: Download Dataset -> Format: **YOLOv8** -> download zip -> unzip here.

After extracting, this folder must look like:

    datasets/helmet_rider/train/images/   + labels/
    datasets/helmet_rider/valid/images/   + labels/
    datasets/helmet_rider/test/images/    + labels/

(You can delete this PLACEHOLDER.md afterwards.)
